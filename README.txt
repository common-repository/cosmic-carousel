== Description ==

Easily create dynamic carousels from every post type registered in your WP. 
Order slides from a draggable list and display each carousel created by a shortcode.
Paste the shortcode in the content area of every page or post where you wanna show the carousel.
 

== Installation ==

1. Go to your admin area and select Plugins â†’ Add New from the menu.
2. Search for "Cosmic Carousel".
3. Click install.
4. Click activate.
5. Navigate to Cosmic Carousel in the admin menu
6. Create a new carousel
7. Copy the generated shortcode (end of the page)
8. Paste the shortcode in the content of the page/post where you wanna show the carousel.

== Frequently Asked Questions ==

= Can I use it to show Woocommerce products? =
- Yes you can select the items from every post type registered in your WP  

= Can I change the order of the item? =
- Yes you can choose and reorder the items from a draggable list
 


== Screenshots ==
